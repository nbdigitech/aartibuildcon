  1. Which of the following statements is/are true in the context of implied addressing mode ?
    S1: All register reference instructions that use only an accumulator follows implied addressing mode
    S2: Zero address instructions in a stack organized computer follows implied addressing mode( Marks: 0.00 )
    1.   S1
    2.   S2
    3.   Both S1 and S2
      Explanation:
      In implied addressing mode the operands are specified implicitly in the definition of the instruction. For example the ‘complement accumulator’ instruction is an implied mode instruction because the operand in the accumulator register is implied in the definition. Zero address instructions in a stack organized computer are implied mode instructions since the operands are implied to be on the top of the stack.
    4.   None of these


  2. In immediate addressing mode, where is the operand placed?( Marks: 0.00 )
    1.   In memory
    2.   In stack
    3.   In instruction
      Explanation:
      In immediate addressing mode the operand is specified in the instruction itself. In other words, an immediate mode instruction has an operand field rather than an address field. The operand field contains the actual operand to be used in conjunction with the operation specified in the instruction. Immediate mode instructions are useful for initializing registers to a constant value.

  7. Consider the following micro operations
    I1: MAR <- (PC)

    I3 : PC <- PC + 1
    I4: IR <- MBR

    1.   I1, I2
    2.   I2,I4
    3.   I3, I4
      Explanation:
      MAR <- PC must be done before MBR <- Memory because memory address stored in MAR which specifies from which location instruction needs to be read. I3 and I4 can be done in single time unit as they are not conflicting.     Interrupt cycle( Marks: 0.00 )

      Explanation:

  9. A machine has a 16-bit architecture. It consist of 64 registers and it support 150 instructions, which have an immediate operand in addition to one register operand. Assuming that the immediate operand is a signed integer, the maximum value of the immediate operand is ?( Marks: 0.00 )

So we can use 1 bit for data. So maximum value we can represent is 1.

  11. A machine has a 32-bit architecture, with 1-word long instructions. It has 128 registers, each of which is 32 bits long. Machine supports 50 instructions, which have an immediate operand in addition to two register operands. Assuming that the immediate operand is an unsigned integer, the maximum value of the immediate operand is( Marks: 0.00 )
    1.   4096
    2.   4095
      Explanation:
      Each instruction has 32 bits. To support 50 instructions, opcode must contain 6-bits. Register operand1 requires 7 bits, since the total registers are 128, Register operand 2 also requires 7 bits. 12-bits are left over for immediate operand. Using 12-bits, we can give maximum 2^12-1=4096-1=4095
    3.   4094
    4.   1024


  12. Consider the following program segment for a hypothetical CPU having three user registers R1, R2 and R3.

    Consider that the memory is byte addressable with size 64 bits, and the program has been loaded starting from memory location 1000 (decimal). If an interrupt occurs while the CPU has been halted after executing the HALT instruction, the return address (in decimal) saved in the stack will be( Marks: 0.00 )
    1.   1070
    2.   1010
    3.   1009
    4.   1072
      Explanation:
      Instruction size is given in words. So first instruction will take 3 words i.e 24 bytes (as 64 bit byte addressable, word size will be 64 bit ) so on for 2nd instruction 16 byte, for 3rd 8 bytes, 4th 16 bytes..5th 8 bytes. As 1st instruction starts from 1000 and the size is 24 bytes so the second instruction starts at the address 1024, likewise 3rd instruction starts at the address 1040, 4th instruction starts at the address 1048, 5th instruction starts at the address 1064 and halt instruction address will be at 1072. As an interrupt occurs executing the HALT instruction, the return address (in decimal) saved in the stack will be the address of the halt instruction. So answer is 1072.


  13. Consider the memory is byte addressable and each word is of size 16 bits. Consider below code segment and the code started from location 1000(decimal).
    ADD R1, R2, R3 ------Instruction size 3 words

    INT

    What is the return address saved in stack if interrupt occurs during the execution of STORE instruction?( Marks: 0.00 )

    2.   1007
    3.   1008

So address for 2nd instruction is 1006. 2nd instruction takes 2 bytes(instruction size is 1 word, 1 word=16 bit that is 2 bytes) so address for the 3rd instruction is 1008.   14. How many memory accesses required by the following instructions?
    DIV r1, r2, @r3
    SUB r1, r2, r3
    MUL r1, r2, (r3)

    Suppose every instruction is one word long, as well as every address.( Marks: 0.00 )
    1.   3
    2.   5
    3.   6
      Explanation:
      SUB r1, r2, r3 require only one memory access, reading the instruction. MUL r1, r2, (r3) require two memory accesses, the first to read the instruction and the other one to read the value from memory location(s), whose address is in r3. DIV r1, r2, @(r3) three memory accesses are made in this case, the first to read the instruction, the second to get M[r3], and the third one to get M[M[r3]]. So total 6 memory accesses are required.
    4.   7


  15. Which of the following things are specified in the indexed mode of instruction?( Marks: 0.00 )

    2.   The value of the index register
    3.   The index value of the current element
    4.   All the above
      Explanation:
      Index Mode for instruction is used to access an array whose elements are in successive memory locations. The content of the instruction code, represents the starting address of the array and the value of the index register, and the index value of the current element. By incrementing or decrementing index register different element of the array can be accessed.
