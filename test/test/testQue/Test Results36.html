  10. Consider the following grammar
    A→int|int+A|int-A|A-(A)
    In context to converting the above grammar into LL(1) grammar which of the following statement/s is/are true?( Marks: 0.00 )
    1.   The grammar is already a LL(1) grammar
    2.   The grammar requires to be left-factored to be converted to LL(1) grammar
    3.   Both left recursion and non-determinism requires to be removed to be converted to LL(1) grammar.
      Explanation:
      Given grammar is:
      A→int|int+A|int-A|A-(A)
      We have to eliminate non-determinism by applying left factoring.
      A→int A' |A-(A), A'→ϵ|+A|-A
      Then eliminating left-recursion the grammar becomes,
      A→int A' A'',
      A'→-(A) A'' |ϵ,
      A'→ϵ|+A|-A
      The above grammar is LL(1).
      Note: We could have also performed first the removal of left recursion followed by left factoring.
    4.   The grammar cannot be converted to LL(1) grammar

      Explanation:
      A and B are Turing Decidable.

      Let MA,MB,MC be TM’s that recognize languages A,B,C, we know that every string will be accepted by exactly one of these machines.
      We use this idea to construct a TM that decides "on input w" .

      3] As soon as MB (or) MC accept. Reject.
