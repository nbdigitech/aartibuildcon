  25. What is the efficient time complexity to find the third largest number in an array containing random elements?( Marks: 2.00 )
    1.   O(logn)
    2.   O(n)
      Explanation:
      Algorithm:
      We can find the third largest in one traversal only.
      1. Initialize first = arr[0] and second = -INFINITY, third = -INFINITY
      2. Iterate the array and compare each element with first
      If arr[i] is greater than all first, second and third:
      third = second
      second = first
      first = arr[i]
      Else compare arr[i] with second, if it is greater than second, then update
      third = second
      second = arr[i].
      Else compare arr[i] with third, if it is greater than third, then update: third = arr[i]
      3. Return third
    3.   O(n log n)
    4.   O(n2)
