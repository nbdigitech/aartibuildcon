  33. Suppose you have a computer system with a 48-bit logical address, page size of 16K and 4 bytes per page table entry. If we have a 48MB program such that the entire program and all necessary page tables are in memory. How much memory is used by program, including its page tables?( Marks: 0.00 )
    1.   40000KB
    2.   49188KB
      Explanation:
      Given 48bit virtual address, 16KB page size and 4 Bytes page table entry, the division into levels of paging is:

      (2^48) / (2^14) = 2^34 entries in the 1st level page table.
      Size of 1st level page table = (2^34) * (2^2) = 2^36 Bytes

      Since 2^36 Bytes is greater than 16KB(2^14), it is further divided into 2nd level paging.

      (2^36)/(2^14) = 2^22 entries in 2nd level page table
      Size of 2nd level page table = (2^22) * (2^2) = 2^24 Bytes

      2^24 Bytes is greater than the page size, so it is further divided into 3rd level page table.

      (2^24)/(2^14) = 2^10 entries in 3rd level page table
      Size of 3rd level page table = (2^10) * (2^2) = 2^12 Bytes. It fits into a single page (2^14 Bytes). So total there are 3 levels of paging and the split is 10 bits for 3rd level PT enties, 12 bits for 2nd level PT enties and 12 bits for 1st level PT enties.


      Since the program size is 48MB, total no of pages required for this program = 48MB/16KB = 3 * 2^10 pages.

      The first level page table itself can hold 2^12 pages = 4 * 2^10 pages.
      So one page 1st level page table is sufficient. Since there are total 3 levels of paging, one 2nd level page table and 1 third level page table are also needed.

      So total size used by the 3 page tables = 2^10 * 4 Bytes + 2^12 * 4 Bytes + 2^12 * 4 Bytes = ( 1 + 4 + 4)* 4 *2^10 Bytes = 9*4*2^10 Bytes = 36KB

      By adding the given program size = 36KB + 48MB = 36KB + 48*1024 KB = (36+49152) KB = 49188 KB
    3.   50000KB
    4.   55000KB
