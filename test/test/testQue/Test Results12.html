  22. Consider the following set of premises
    H1:A→B,B→C,C→~D,A→D,A
    H2:P→Q,~(Q∧~R),~R,P
    Which of the following is true?( Marks: 0.00 )
    1.   Only H1 is inconsistent
    2.   Only H2 is inconsistent
    3.   Both H1and H2are inconsistent
      Explanation:
      H_1:The given premises are
      (1) A→B
      (2) B→C
      (3) C→~D
      (4) A→D
      (5) A
      Proof:
      (6) D from (4),(5) Modus Ponens
      (7) A→C from (1),(2) Transitivity
      (8) A→~D from (7),(3) Transitivity
      (9) ~D from (8),(5) Modus Ponens

      (2) ~(Q∧~R)

      (5) Q→R from (2)

      (7) R from (4),(6) Modus Ponens
