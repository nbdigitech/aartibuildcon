      Let C be the language obtained by adding B to A i.e., C=A∪B. Suppose that C is regular and we now show this is impossible since A∩B=∅.
      We have that A=C-B since C & B are regular. As regular languages are closed under differs A also is a Regular language but we assumed that A is non-regular which is a contradiction. Hence A must be regular.

  15. S → Aa | b
    A → Ac | Sd | ∈
    The equivalent production rules corresponding to the above production rules:( Marks: 0.00 )
    1.   S → A’aS’ | bS’
      S’ → dA’aS’ | ∈
    2.   S → A’aS’ | bS’
      S’ → dA’aS’ | ∈
      A’ → cA’ | ∈
    3.   S → A’aS’ | bS’
      S’ → dA’aS’ | ∈
      A → SdA’ | A’
      A’ → cA’ | ∈
      Explanation:
      The given grammar is left recursive.
      A grammar is said to be left recursive if it is of the form:
      A → A α | β
      The grammar after left recursion is:
      A → β A'
      A' → ∈ | α A'
      So for the given grammar the productions after removal of immediate left recursion is :
      S → A’aS’ | bS’
      S’ → dA’aS’ | ∈
      The above grammar is not free of all left recursion. The resulting grammar is:
      S → A’aS’ | bS’
      S’ → dA’aS’ | ∈
      A → SdA’ | A’
      A’ → cA’ | ∈
