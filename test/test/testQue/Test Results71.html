  1. In a IPv6 protocol communication, If total Router Header length is 448 bits. Then how many routers or hops are there for the packet to reach destination.( Marks: 0.00 )
    1.   3
      Explanation:
      Router header length = 448 / 8 = 56 bytes
      First 8 bytes are for Next header(1 byte), Header Extension length(1 byte), Routing type(1 byte), segments left(1 byte), Reserved(4 bytes).
      So 56 - 8 = 48 bytes are hop addresses.
      Each address is 16 bytes. So 48 / 16 = 3
    2.   4
    3.   5
    4.   2

    1.   20
    2.   40
    3.   45
      Explanation:

      looks like we need 10 × 9 keys. However, if the same key can be used in both
