  1. Let p and q be the propositions “Swimming at the New Jersey shore is allowed” and “Sharks have been spotted near the shore” respectively. How can this English sentence be translated into a logical expression.
    “If sharks have not been spotted near the shore, then the swimming at the New Jersey shore is allowed”.( Marks: 1.00 )
    1.   p→~q
    2.   ~q→p
      Explanation:
      Let p represents “Swimming at the New Jersey shore is allowed”.
      Let q represents “Sharks have been spotted near the shore”.
      ~qrepresents “It is not the case that sharks have been spotted near the shore”.
      This ~q can be more simply expressed by “Sharks have not been spotted near the shore”.
      Then the sentence can be translated to
      (~q→p)
    3.   ~q→~p
    4.   p↔q

  5. Consider the premises
    {P,(P→(Q⋁R) ),(Q→~P) }
    Which of the following is a logical consequence of the above premises?( Marks: 0.00 )
    1.   Q
    2.   ~P
    3.   ~R
    4.   R
      Explanation:
      The premises are
      (1) P
      (2) P→(Q⋁R)
      (3) Q→~P
      (4) ~Q (from 1 & 3) (Modus tollens)
      (5) Q⋁R (from 1 & 2) (Modus Ponens)
      (6) R (from 4 and 5) (Disjunctive syllogism)


  6. Let p,q and r be the propositions.
    p: You get an A on the final exam.
    q: You do every exercise in this book.
    r: You get an A in this class.
    How can this English sentence are translated into a logical expression.
    “Getting an A on the final and doing every exercise in this book is sufficient for getting an A in this class”.( Marks: 2.00 )
    1.   p→(q⋀r)
    2.   (p⋀q)→~r
    3.   (p⋀q)→r
      Explanation:
      The logical expression for the “Getting on A on the final and doing every exercise in this book” is p⋀q.
      r: You get an A in this class.
      If (p⋀q) is true then 'r' is true.
      If (p⋀q) is false then 'r' may be true or may not be true.
      So, the logical expression for the sentence is : ( p⋀q)→r.
    4.   (p⋁q)→r


  7. Consider the following set of premises
    H1:P→(Q→R),S→(Q⋀~R),P⋀S
    H2: ~A⋁B,~B,A
    Which of the following is true?( Marks: 0.00 )
    1.   Only H1 is inconsistent
    2.   Only H2 is inconsistent
    3.   Both H1 and H2 are inconsistent
      Explanation:
      If the false conclusion is derived from the given premises then the premises are inconsistent.
      H1: The given premises are
      (1) P→(Q→R)
      (2) S→(Q⋀~R)
      (3) P⋀S
      From (3), by simplification rule we have
      (4) P
      (5) S
      (6) Q→Rfrom(1), (4) [Modus Ponens]
      (7) Q⋀~Rfrom(2), (5) [Modus Ponens]
      (8) ~(~Q⋁R)from(7) [De morgan’s law]
      (9) ~(Q→R)from(8)
      (10) (Q→R)⋀~(Q→R)from(6), (9) [Conjunction]
      (11) F from(10)
      ∴H1 is inconsistent.
      H2: The given premises are
      (1) ~A⋁B
      (2) ~B
      (3) A
      From (1), (2) Disjunction syllogism
      (4) ~A
      (5) A⋀~A (3), (4) Conjunction
      (6) F
      ∴H2 is inconsistent
    4.   None of these


  8. Which of the following are valid arguments?
    A1:[P→(Q→R),Q]⟹(P→R)
    A2:[(P→Q),(~Q→R),~R]⟹P( Marks: 0.00 )
    1.   A1 is valid and A2 is not valid
      Explanation:
      From CP rule, the given premises in A1 are
      A1: 1. P→(Q→R)
      2. Q
      3. P
      4. Q→Rfrom(1) and (3) [Modus Ponens]
      5. R from (2) and (4) [Modus Ponens]
      ∴P→Ris the conclusion derived from the given premises. So, A1 is valid.
      A2: 1. P→Q
      2. ~Q→R
      3. ~R
      4. Qfrom (2), (3) Modus Ponens
      We cannot combine (1) and (4) by any rule to get the conclusion.
      So, the argument A2 is not valid.
    2.   A1 is invalid and A2 is valid
    3.   Both A1 and A2 are valid
    4.   Both A1 and A2 are invalid

  10. Which of the following is invalid?( Marks: 0.00 )
    1.   (p→(q⋁r) )⟺((p⋀~q)→r)
    2.   ((p→r)⋀(q→r)⟺((p⋁q)→r)
    3.   p→(q→r)⟺q→(p→r)
    4.   (p↔q)⟺(p⋁q)⋁(p⋀q)

      ∴(p→(q⋁r) )⟺((p⋀~q)→r)
      Option (b);
      (p→r)⋀(q→r)⟺(~p⋁r)⋀(~q⋁r)
      ⟺(~p⋀~q)⋁r [∵ De morgan’s law]
      ⟺~(p⋁q)⋁r
      ⟺(p⋁q)→r

      p→(q→r)⟺~p⋁(q→r)

      ⟺~q⋁(~p⋁r)

  11. Let I(x) be the statement “x has an Internet connection” where the domain for the variable x consists of all students in your class. Which one of the following statements is (are) true?
    Statement I: The logical translation of the statement “Someone in your class does not have internet connection” is ∃x (~I(x) ).
    Statement II: The logical translation of the statement “Not everyone in your class has an internet connection” is ~∀x I(x).( Marks: 0.00 )
    1.   Statement I is true
    2.   Statement II is true
    3.   Both I and II are true
      Explanation:
      Given (x) : x has an internet connection
      ~I(x) : x doesn’t have an internet connection
      So, the meaning of ∃x~ (I(x) ) is “There is a person who does not have an internet connection”.
      The above sentence can be written as “Someone in your class does not have an internet connection”.
      ⟹The given statement is “Not everyone in your class has an internet connection”.
      The logical translation of the statement “everyone in your class has an internet connection” is ∀x I(x).
      The negation of ∀x I(x) is “It is not the case that everyone in your class has an internet connection”.

      ~∀x I(x)is “Not everyone in your class has an internet connection”.
    4.   Both I and II are false


  12. Suppose the predicate F(x,y) is used to represent the statement that person x can fool person y. Which one of the following statements below expresses the best meaning of the formula ~∃x ∀y F(x,y)?( Marks: 0.00 )
    1.   everyone can fool somebody
    2.   no one can fool everyone
      Explanation:
      Given F(x,y):x can fool y
      (a) every person x can fool at least one person y. So, the mathematical logic is
      ∀x ∃y F(x,y)
      (b) The negation of “at least one” is “no one”.
      ∴ The new statement is the negation of “No one can fool everyone” .
      So the new statement is “at least one person can fool everyone”.
      The mathematical logic for the statement “at least one person can fool everyone” is
      ∃x ∀y F(x,y)
      ∴The negation of ∃x ∀y F(x,y) is “No one can fool everyone”.
      i.e. ~∃x ∀y F(x,y)
      (c) The mathematical statement for
      “everyone can fool somebody” is ∀x ∃y F(x,y)
      But the meaning of ∀y ∃x F(x,y) is everyone can be fooled by somebody.
    3.   everyone can be fooled by somebody
    4.   none of these


  13. What is the logical translation of the following statement?
    “There are some people who are not software engineers but are developers”( Marks: 0.00 )
    1.   ~∃x (S(x)⋀D(x) )
    2.   ∀x (S(x)→D(x) )
    3.   ∃x (~(S(x))⋀D(x) )
      Explanation:
      Let the Domain for the variable ‘x’ consists of all people
      S(x): x is a software engineer
      D(x): x is a Developer
      Option A:
      The meaning of ∃x (S(x)⋀D(x) ) is atleast one person is a software engineer and a developer.
      The negation of ∃x (S(x)⋀D(x) )⟺~∃x (S(x)⋀D(x) )⟺∀x (~S(x)⋁~D(x) )
      Every person is not a software engineer or not a developer.
      Option B:
      The meaning of S(x)→D(x) is
      “If x is a software engineer then x is a Developer”.
      ∀x (S(x)→D(x) ) : all software engineers are developers.
      Option C:
      “but” means “and”
      ~S(x):x is not a software engineer
      ~S(x)⋀D(x):x is not a software engineer and x is a developer
      The meaning of ∃x (~S(x)⋀D(x) ) is
      “There exists at least one person who is not a software engineer but a developer”.
      This means, “There are some people who are not software engineers but are developers”.
      Option D:
      ~D(x):x is not a developer
      S(x)⋀~D(x):x is a software engineer and x is not a developer
      The meaning of ∃x (S(x)⋀~D(x) ) is
      “There are some software engineers who doesn’t know development”.
    4.   ∃x (S(x)⋀~D(x) )

  15. Which of the following is not logical implication?( Marks: 0.00 )
    1.   P⋀Q⟹P↔Q
    2.   P↔Q⟹Q→P
    3.   P↔Q⟹P→~Q
      Explanation:
      If P⟹Q means P→Q is a tautology
      If P→Q is not a tautology that means it is not a logical implication
      Option (a):
      P⋀Q(LHS)⟹P↔Q(RHS)
      Whenever P and Q is True, the LHS is true and RHS is also true.
      (If LHS is True for some P and Q then RHS must be True for the same P and Q)
      So, P⋀Q⟹P↔Q is a logical implication.
      Option (b):
      Whenever bi-conditional is True, conditional is also True.
      P↔Q⟹Q→P
      Option (c):
      If P and Q is True then P↔Q is true. But P→~Q is False. So P↔Q⟹P→~Q is not a Tautology.
      ∴P↔Q⟹P→~Q is not a logical implication.
      Option (d):
      If P is True and Q is False. Then ~(P→Q) is true. If Q is false then ~Q true.
      So, ~(P→Q)⟹~Q is a logical implication.
    4.   ~(P→Q)⟹~Q
