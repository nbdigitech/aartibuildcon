  10. An array CS[19][18] is stored in row major order with each element occupies 4 bytes of memory, and the address of CS[7][5] is stored at the location 1600. Find the address of CS[12][14]. Assume array index starting with ‘1’( Marks: 2.00 )
    1.   1684
    2.   1884
    3.   1996
      Explanation:
      Array , words per element is 4 bytes
      Given address of CS[7][5] is 1600
      By using below formula we can get base address
      Address of CS[7][5] = Base + 4 * [N(i – 1) + (j – 1) ]
      1600 = Base + 4 * [18 * (7 – 1) + (5 – 1)]
      1600 = Base + (18 * 6 + 4)
      Base = 1600 – 448
      Base = 1152
      Now to find address of CS[12][14]
      CS[i][j] = Base + 4 * [N(i – 1) + (j – 1)]
      = 1152 + 4 * [18 * (12 – 1) + (14 – 1)]
      = 1152 + 4 * (18 * 11 + 13)
      = 1152 + 4 * (198 + 13)
      = 1152 + 4 * 211
      = 1152 + 844
      CS[12][14] = 1996
    4.   1784

    1.   1707
    2.   1565
    3.   1808
      Explanation:

      Address A[p][q] = Base + W * [p + M * q],
      Array M[25][20], words per element = 4 bytes
      Given M[4][5] address = 1500
      ⇒ 1500 = Base + 4 * [(4 – 0) + 25(5 – 0)]
      1500 = Base + 4 * [4 + 125]
      1500 = Base + 4 * 129
      Base = 1500 – 516 = 984
      Now, to find M[6][8]
      ⇒ 984 + 4 * [(6 – 0) + 25(8 – 0)]
      984 + 4 * [6 + 200]
      984 + 4 * 206
      984 + 824
      M[6][8] = 1808
    4.   1926

  14. In array p[5][5], the address of p[1][1] and p[2][1] are 5000 and 5010 respectively and each element occupies 2 bytes of memory. Then the array has been stored in ___ order, assume that index starts with p[1][1]( Marks: 2.00 )
    1.   Row Major order
      Explanation:
      Address of p[1][1] is 5000
      p[2][1] is 5010
      For example, let the matrix is stored in RMO, then to access an element in RMO, formula is
      L0 + [(i – 1) r2 + (j – 1) ] * c
      For p[2][1] = 5000 + [(2 – 1)(5) + (1 – 1)] * 2
      5000 + (5(2)) = 5000 + 10 = 5010.
      for the same example, let the matrix is stored in CMO, then to access an element in CMO, formula is
      L0 + [(i – 1) + r1*(j – 1) ] * c
      For p[2][1] = 5000 + [(2 – 1) + 5*(1– 1)] * 2
      5000 + 2= 5000 + 2= 5002.
      Clearly it is not stored in CMO, so options b & c are wrong
    2.   Column Major order
    3.   Both Row Major and Column Major order
    4.   None of these
