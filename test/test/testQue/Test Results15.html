  41. A Turing machine that does proper subtraction has how many states?
    (Note: Proper subtraction is defined to be m – n for m≥n and zero for m<n)( Marks: 0.00 )
    1.   7
      Explanation:
      Let ‘M’ be the TM, which can proper subtraction. The TM starts with 0^m 10^n on its tape, halts with 0^(m-n) on its tape. M respectively replaces its leading 0 by blank, then searches for a 1 followed by a 0 by blank, then searches for a 1 followed by a 0 and changes the 0 to 1. Next, M moves left until it encounters a blank and then repeats the cycle. The repetition ends if
      i. Searching right for a 0, M encounters a blank. Then, the n 0’s in 0^m 10^n have all been changed to B, M replaces the n+1 1’s by a 0 and n B’s leaving m-n 0’s on its tape.
      ii. Beginning the cycle, M cannot find a 0 to change to a blank, because the first m 0’s already have been changed. Then n≥m, so m-n=0, M replaces all remaining 1’s and 0’s by B.

    2.   O(nlogn)
    3.   O(n)
      Explanation:

    1.   8 Bytes
      Explanation:
      We need the page table to be fitting into a single page. So the page table size has a maximum value of page size.

      So, page table size <= page size.
      Page size = 32MB = 2^(25) Bytes

      page table size = No. of pages * page table entry size

      No. of pages = Virtual Address Space / page size = 128 TB / 32MB = 2(22)
      Let page table entry size = e

      Page table size = 2(22) * e

      2(22) * e <= 2(25)
      e <= 2(3)
      e <= 8 Bytes
      So, the maximum value for page table entry size is 8 Bytes.
