  14. Consider the following statement(s):
    P: Cannonical cover may not be unique
    Q: F = {AB → C, A → B, B → A}. Cannonical cover of F is not unique.

    Which of the above statement is true?( Marks: 1.00 )
    1.   Both P and Q true
      Explanation:
      We know that the given FD set may have more than one cannonical cover. So, P is correct.
      Given F = {AB → C, A → B, B → A}
      Step 1: Decompose FDs such that RHS of an FD contain single attribute
      Need not perform as RHS of all FDs contain single attribute.
      Step 2: Remove if any Redundant FDs are present
      AB → C → (1)
      A → B → (2)
      B → A → (3)
      Checking whether FD (1) is redundant or not:
      Find (AB)+ from remaining FDs
      (AB)+ = AB
      (∵ AB does not determines C) FD (1) is not redundant
      Checking whether FD (2) is redundant or not:
      Find (A)+ from remaining FDs
      (A)+ = A
      (∵ A does not determines B) FD (2) is not redundant
      Checking whether FD (3) is redundant or not:
      Find (B)+ from remaining FDs
      (B)+ = B
      (∵ B does not determines A) FD (3) is not redundant
      Step 3: Find the redundant attributes on LHS and delete them. Only LHS of FD (1) has 2 attributes. So, if we check for FD (1), it is sufficient.
      Check whether A can determine B with the help of FDs other than FD (1)
      (A)+ = AB
      (∵ A can determine B) FD (1) can be written as A → C
      Check whether B can determine A with the help of FDs other than FD (1)
      (B)+ = BA
      (∵ B can determine A) FD (1) can be written as B → C
      So, they are two minimal covers for F
    2.   P is true and Q is false
    3.   P is false and Q is false
    4.   Both P and Q are false.

  48. Consider the following problems about Turing machines:
    S1: To determine, given a Turing machine M, a state q, and a string w, whether M ever reaches state q when
    started with input w from its initial state.
    S2: To determine, given a Turing machine M and a string w, whether M ever moves its head to the left when started with input w.
    Which of the above statement/s is /are solvable?( Marks: -0.66 )
    1.   S1 is solvable, S2 is not solvable
    2.   S1 is not solvable, S2 is solvable
      Explanation:
      Justification for statement S1:
      To determine, given a Turing machine M, a state q, and a string w, whether M ever reaches state q when
      started with input w from its initial state. This is not solvable.
      Essentially, if we can tell whether a machine M ever reaches some state q, then let q be M's halt state (and we can design M so that it has only one halt state). If it ever gets to q, it must have halted. More formally:
      L1 = H = {s = "M" "w" : M halts on input string w}
      ↓ T
      (?M2) L2 = {s : "M" "w" "q" : M reaches state q when started with input w from its initial state}

      Let T create, from M the machine M* as follows:
      Initially M* equals M. Next, a new halting state H is created in M*. Then, from each state that was a halting state in M, we create transitions in M* such that for all possible values of the current tape square, M* goes to H. We create no other transitions to H. Notice that M* will end up in H in precisely the same situations in which M halts.
      Now let T ("M" "w") = T '("M") "w" "H"
      So, if M2 exists, then M1 exists. It invokes T' to create M*. Then it passes "M*", "w", and "H" to M2 and returns whatever M2 returns. But M1 doesn't exist. So neither does M2.

      Justification for statement S2:
      To determine, given a Turing machine M and a string w, whether M ever moves its head to the left when
      started with input w. This one is solvable. We will assume that M is deterministic. We can build the deciding
      machine D as follows. D starts by simulating the operation of M on w. D keeps track on another tape of each
      configuration of M that it has seen so far. Eventually, one of the following things must happen:
      1. M moves its head to the left. In this case, we say yes.
      2. M is stuck on some square s of the tape. In other words, it is in some state p looking at some square s on the tape and it has been in this configuration before. If this happens and M didn't go left yet, then M simply hasn't moved off of s. And it won't from now on, since it's just going to do the same thing at this point as it did the last time it was in this configuration. So we say no.
      3. M moves off the right hand edge of the input w. So it is in some state p looking at a blank. Within k steps (if k is the number of states in M), M must repeat some state p. If it does this without moving left, then again we know that it never will. In other words, if the last time it was in the configuration in which it was in state p, looking at a blank, there was nothing to the right except blanks, and it can't move left, and it is again in that same situation, it will do exactly the same thing again. So we say no.
    3.   Both S1 and S2 is solvable
    4.   Neither S1 nor S2 is solvable
