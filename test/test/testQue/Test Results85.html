  5. Given that f(a,b,c)=Σ(0,1,5). How many covering functions does f have other than itself?( Marks: 0.00 )
    1.   5
    2.   8
    3.   32
    4.   31
      Explanation:
      f(a,b,c)=Σ(0,1,5)
      It has 3 min-terms. Out of 8 min-terms, 5 min-terms are left. Any function with terms (0,1,5) and additional terms is a covering function for f, and f itself covers f.
      0-f
      1-f
      2⟶2C1
      3⟶2C1
      4⟶2C1
      5-f
      6⟶2C1
      7⟶2C1
      ⟹2×2×2×2×2=32
      If no min-terms out of 5 is taken, then it is f itself.
      We need to eliminate this.
      ⟹32-1=31
