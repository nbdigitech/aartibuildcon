  12. What is the worst case time complexity to construct a binary search tree when inorder an preorder traversal of the tree is given?( Marks: 0.00 )
    1.   O(n)
    2.   O(n log n)
    3.   O(n2)
      Explanation:
      Algorithm:
      Step 1: Select an element from preorder. Increment the preorder index variable to select next element in next recursive call.
      Step2: Create a new tree node temp with the data as selected element.
      Step 3: Find the selected element’s index in inorder. Let the index be inIndex
      Step 4: Call createTree for elements before inIndex and make the create tree as left subtree of temp
      Step 5: Call createTree for elements after inIndex and make the create tree as right subtree of temp
      Step 6: return temp
      Worst case it takes O(n2) i.e left skewed or right skewed binary search tree
    4.   O(n3)

  24. Which of the following statements are False?
    I.The depth of a breadth-first search tree on an undirected graph G=(V,E) from an arbitrary vertex v∈V is the diameter of the graph G. (The diameter d of a graph is the smallest d such that every pair of vertices s and t have δ(s, t) ≤ d.)
    II.Every directed acyclic graph has exactly one topological ordering.
    III. Given a graph G = (V, E) with positive edge weights, the Bellman-Ford algorithm and Dijkstra’s algorithm can produce different shortest-path trees despite always producing the same shortest-path weights.( Marks: 0.00 )
    1.   I and II
      Explanation:
      I. False. An arbitrary vertex could lay closer to the ’center’ of the graph, hence the BFS depth will be underestimating the diameter. For example, in graph G = (V, E) = ({a, v, b}, {(a, v),(v, b)}), a BFS from v will have depth 1 but the graph has diameter 2.
      II. False. Some priority constraints may be unspecified, and multiple orderings may be possible for a given DAG. For example a graph G = (V, E) = ({a, b, c}, {(a, b),(a, c)}) has valid topological orderings [a, b, c] or [a, c, b]. As another example, G = (V, E) = ({a, b}, {}) has valid topological orderings [a, b] or [b, c].
      III. True. Both algorithms are guaranteed to produce the same shortestpath weight, but if there are multiple shortest paths, Dijkstra’s algorithm will choose the shortest path according to the greedy strategy, and Bellman-Ford will choose the shortest path depending on the order of relaxations, and the two shortest path trees may be different.
    2.   I and III
    3.   II and III
    4.   I, II and III
