  6. Consider the following languages.
    L1 = {aibjck | i≠j or j≠k or i≠k}.
    L2 = {xyz│x,y,z∈{0,1}*,|x|=|z| and n0(x) ≥ n0(z)}.( Marks: -0.66 )
    1.   L1 is CFL but L2 is not
    2.   L1 and L2 are CFL
    3.   L1 is regular and L2 is CFL
    4.   L1 is CFL and L2 is regular
      Explanation:
      L1 is context free language as we just need one stack for accepting the strings in the language.
      L2 = {xyz│x,y,z∈{0,1}*,|x|=|z| and n0(x) ≥ n0(z)}.
      In this language, y can eat up everything except the first and last character in the string. The first character should always be 0 and z can be either 0 or 1. This language can be represented by a regular expression 0(0+1)*(0+1)+ε+1(0+1)*1. Therefore this is regular.


  7. Consider the two grammars G1 & G2 as given below

    where S is the start symbol in each grammar.
    Let L(G1) be L1, L(G2) be L2 and L(G1) ∩ L(G2) = L3
    Which of the following is CORRECT?( Marks: 2.00 )

  8. For each of the following languages, determine whether it is (I) regular, (II) not regular but context-free, or (III) not context-free.
    L1={ai bj ck│i<j<k}
    L2={ai bj ck│i<j or k<j}
    L3={ai bj ck│i<j and k<j}
    L4={ai bj ck│i<j ⇒k<j}
    L5={ai bj ck |i,j,k≥0}( Marks: -0.66 )
    1.   L1: (II),L2: (III),L3: (III), L4: (III), L5: (I)
    2.   L1: (III),L2: (II),L3: (III), L4: (II), L5: (I)
      Explanation:
      For strings in the language L1={ai bj ck│i<j<k}, we need to compare number of a’s, b’s and c’s simultaneously which require more than one stack to recognize. So, it cannot be CFL. So, it is (III)
      For strings in the language L2={ai bj ck│i<j or k<j}, we need to compare either number of a’s and b’s or number of b’s and c’s. This can be done using one stack. So, it is CFL. To be precise, it is non-deterministic CFL as we need to guess initially whether we are comparing a’s and b’s or b’s and c’s. So, it is (II)
      For strings in the language L3={ai bj ck│i<j and k<j}, we need to compare number of a’s, b’s and c’s simultaneously which require more than one stack to recognize. This is same as L1 which is not CFL. So, it is (III)
      For language L4={ai bj ck│i<j ⇒k<j}, the condition i<j ⇒k<j can be written as (i<j)' or k<j which is equivalent to i≥j or k<j.
      ∴L4={ai bj ck│i≥j or k<j} is similar to L2 which require only one stack to recognize the strings in the language. So, L4 is (II)
      L5 is regular. (I)
    3.   L1: (III),L2: (III),L3: (III), L4: (III), L5: (I)
    4.   L1: (II),L2: (II),L3: (II), L4: (III), L5: (I)


  9. G = (V, T, P, S), where V = {S, W, X, Y, Z}, T = {a, b, c}, S is the start variable, and the productions P are
    S →WX|Y
    W→ cWa|ε
    X → bX|ε
    Y → cYb|Z
    Z → aZ|ε

    Which of the following languages is the correct language generated by the above grammar?( Marks: -0.66 )
    1.   L = {ci aj bk | i, j, k ≥0, and i = j or i = k}
      Explanation:
      The given grammar is
      S →WX|Y
      W→ cWa|ε
      X → bX|ε
      Y → cYb|Z
      Z → aZ|ε
      The start symbol S derives two types of strings. We can see this from the production S →WX|Y.
      W derives the strings having equal number of c’s and a’s and a is never followed by c. X derives b*. Whereas, Y derives the strings having equal number of c’s and b’s and b is never followed by c. Z derives a*.
      Observe that S can derive empty string also.
      Therefore, S derives strings of the form ci aj bk having either equal number of c’s and a’s or equal number of a’s and b’s, where i, j, k ≥0.

      ∴ L = {ci aj bk | i, j, k ≥0, and i = j or i = k}
    2.   L = {ci aj bk | i, j, k ≥0, and i > k or j < i}
    3.   L = {ci aj bk | i, j, k ≥0, and i = j or j = k}
    4.   L = {ci aj bk | i, j, k >0, and i = j and i = k}

  11. Which of the following grammars are equivalent?( Marks: 0.00 )
    1.   G1: S → XY, X → xXX|ε , Y → yxYY|ε
      G2: S → ZY|Y|ε , Z → xZZ|xZ|x, Y → yxYY|yxY|yx
    2.   G1: S → XY, X → xXX|ε , Y → yxY|ε
      G2: S → ZY|Z|Y, Z → xZZ|xZ|x, Y → yxYY|yxY|yx
    3.   G1: S → SaSQaS|ε, Q → cS
      G2: S → SaPaS|ε, P → ScS
      Explanation:
      G1: S → XY, X → xXX|ε , Y → yxY|ε
      G2: S → ZY|Z|Y, Z → xZZ|xZ|x, Y → yxYY|yxY|yx
      G1 generates ε but G2 does not. So, they are not equivalent.

      G1: S → XY, X → xXX|ε , Y → yxYY|ε
      G2: S → ZY|Y|ε , Z → xZZ|xZ|x, Y → yxYY|yxY|yx
      G1 generates x but G2 does not.

      G1: S → PQ|p, P → q
      G2: S → p|q
      G1 generates p but G2 generates {p, q}

      G1: S → SaSQaS|ε, Q → cS
      Q can be eliminated by substituting Q → cS in S → SaSQaS|ε. This results in the grammar S → SaScSaS|ε
      G2: S → SaPaS|ε, P → ScS
      P can be eliminated by substituting P → ScS in S → SaPaS|ε. This results in the grammar S → SaScSaS|ε
      So, both the grammars define the same language.
    4.   G1: S → PQ|p, P → q
      G2: S → p|q

  15. Linked Answer Questions Q14 & Q15
    Here are the transitions of a deterministic pushdown automata. The start state is qo, f is the accepting state.

    Σ = {a, b}
    τ = {0, 1, z0}
    Note:
    Where in states column (q0-z0) represents current state is q0, top of the stack is z0 and goes to state q1 and pushes two 0's onto the stack.

    Which of the following language is accepted by the above PDA?
    (na=number of a's in w, nb=number of b's in w)( Marks: 0.00 )
    1.   L={w | w ∈ {a, b}* and na(w) = nb(w)}
    2.   L={w | w ∈ {a, b}* and 2na(w) = nb(w)}
      Explanation:
      If we observe column 'a',
      In most of the cases, for every one 'a' seen on input tape we are pushing two 0's to the stack[i.e.,(q1,00z0),(q1,000)] and for every one 'b' seen on the input tape we are pushing single 1 onto the stack.
      (NOTE: Those two obsevations are only for push operation)
      Consider the pop operations now,
      for 'a' seen on the input tape with current top of stack as 1 we are using pop operation and
      for 'b' seen on the input tape with current top of stack as 0 we are using pop operation.

      ∴ If we see input tape symbol 'a' with top of stack 0 we are pushing to stack or if top of stack is 1, we pop out from the stack.

      As mentioned above, for 'a' two 0's are pushed onto the stack and for 'b' one 1 is pushed.This says that, Number of b's are twice as number of a's.
    3.   L = {w | w ∈ {a,b}* and na(w) = 2nb(w)}
    4.   L={anbn | n ≥ 0 }
