  1. Consider an ethernet of distance between 2 peers is 500 meters. Is it possible to increase the length of this ethernet to multiple segments, if a repeater is added at end of each segment to connect to another segment?( Marks: -0.33 )
    1.   No
      Explanation:

In fact, repeaters are a part of current Ethernet standard, which specifies that the network will not operate correctly if more than four repeaters separate any pair of station.

  7. Assume there is a satellite communication link between moon to earth with a capacity of 1-Mbps. The communication is happening between moon and earth with 1000-bit frames and takes 270-ms one way delay. What is the maximum link utilization, if we use
    Stop-and-wait flow control
    Continuous flow control with a window size of 7?( Marks: 2.00 )
    1.   1→ 0.017, 2 → 0.002
    2.   1→ 0.02, 2 → 0.0013
    3.   1→ 0.002, 2 → 0.013
      Explanation:
      For stop-and-wait: Utilization = 1/(1+2a)
      Continuous flow control(SR or GO-back-N):Utilization = W/(1+2a)
      Where , a = (Propagation Time)/ (Transmission Time)
      Now, Transmission Time = (Frame Length) / (Data Rate)
      The following values are given:
      Propagation Time = 270 ms = 270 x 10-3sec
      Frame Length = 1000 bits
      Data Rate = 1 Mbps= 1 x 106 bps
      Therefore, a = (270 x 10-3)/ (1000 / 106) =270
      1.U= 1/(1+2a) = 1/(1+2 x 270) = 1/541 = 0.002
      2.U= W/(1+2a) = 7/(1+2 x 270) = 7/541 = 0.013
    4.   1→ 0.012, 2 → 0.022

  10. Consider a scenario of GO-BACK-4 protocol with 10 bits sequence numbers. Assume that s is the sequence number of subsequent packet at time t0, Then what can be the possible range of window size at sender. Note: medium does not change the order of packets.( Marks: -0.66 )
    1.   [s-9, s-1]
    2.   [s,s+3]
      Explanation:
      Here we have a window size of N=4.
      We have 2 cases possible here,
      First case, Suppose the receiver has received packet s-1, and has acknowledged that and all other preceding packets. If all of these ACK's have been received by sender, then the sender's window is [s, s+N-1].
      Second case, Suppose next that none of the ACKs have been received at the sender. the sender's window contains s-1 and the N packets up to and including s-1. The sender's window is thus [s-N,s-1].
    3.   [s-4, s-9]
    4.   [s-4,s]
