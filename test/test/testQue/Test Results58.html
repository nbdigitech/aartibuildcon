  5. If R1,R2,R3 and R4 are relations on a set A={x,y,z} then which of the following statements is false.( Marks: 1.00 )
    1.   R1={(x,x),(y,y),(z,z),(x,z)}is both reflexive and anti-symmetric
    2.   R2={(x,x),(y,y),(x,z) }is anti-symmetric but not symmetric
    3.   R3={(x,z),(z,x),(x,x) } is not transitive and not irreflexive
    4.   R4={(x,x),(y,y),(x,z),(z,x) } is both reflexive and symmetric
      Explanation:
      A relation ‘R’ on set A is said to be
      (i) Reflexive if (x,x)∈R ∀x∈A
      (ii) Irreflexive if (x,x)∉R ∀x∈A
      (iii) Symmetric if (x,y)∈R then (y,x)∈R ∀x,y∈A
      (iv) Antisymmetric if (x,y)∈R and (y,x)∈R then x=y ∀x,y∈A
      (v)Asymmetric if (x,y)∈R then (y,x)∉R ∀x,y∈A
      (vi) Transitive if(x,y)∈R and (y,z)∈R then (x,z)∈R ∀x,y,z ∈A
      a) Given R1={(x,x),(y,y),(z,z),(x,z)}
      R1contains{(x,x),(y,y),(z,z) } so it is reflexive.
      R1has(x,z) but not (z,x). So it is antisymmetric.
      ∴R1 is both reflexive and antisymmetic.
      b) Given R2={(x,x),(y,y),(x,z) }
      In the relation R2, (x,z) is present but not (z,x). So, the relation is antisymmetric but not symmetric.
      c) Given R3={(x,z),(z,y),(x,x) }
      (x,x)is present in the relation R3. So, R3 is not irreflexive. (x,y) is not present in the relation so the relation not transitive.
      d) Given R4={(x,x),(y,y),(x,z),(z,x) }
      (z,z)is not present in the relation R4. So the relation R4 is not reflexive.
      ∴R4 is false.
