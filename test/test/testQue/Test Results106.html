  12. Consider a system TLB lookup takes 30ns and main memory takes 200ns what will be the TLB miss rate if effective memory access time is 250 ns?( Marks: 0.00 )
    1.   30%
    2.   20%
    3.   40%
    4.   10%
      Explanation:
      Sol:
      TLB’s access time = 30ns
      Memory access time = 200ns
      EMAT = 250ns
      EMAT = TLBhit(TLBaccess+MainMemoryaccesstime)+TLBmiss*(TLBaccess+2*MainMemoryaccesstime)
      250=h1*230+(1-h1)(30+400)
      200h1 = 430-250
      200h1= 180
      h1 = 90%

      So, miss rate is 10%
