  7. Consider a computer system in which cache memory write hit takes 10ns and and miss takes 100ns. Cache memory read hit takes 5ns and miss takes 55ns. The cache is having 90% hit. The system received 1000 fetch instructions out of which, 500 operand fetch operations and 500 operand write operations. The average time taken to execute above 1000 instructions is___________?( Marks: 0.00 )
    1.   25750ns
      Explanation:
      Given
      cache memory write hit takes(twh) 10 ns,
      cache memory write miss takes (twm) 100 ns,
      cache memory read hit takes (trh) 5 ns,
      cache memory read miss takes (trm) 55 ns
      and 90% hit rate
      Average access time for write operation(Twrite) = twh+(1-h1) twm
      = 10+(0.1)100
      = 20 ns


      Average access time for read operation(Tread) = trh+(1-h1) trm
      = 5+(0.1)55
      = 10.5
      Out of 1000 fetch operations 500 are operand fetch(read instructions) instructions and 500 are operand write instructions.


      Total time taken to execute 1000 instructions = 1500(10.5) + 500(20)
      = 25750 ns.

  15. Consider a system with CPI of 1.0 on a 5 GHz machine with a 2% miss rate and memory access time of 100ns. To reduce miss penalty designers decided to add a L2 cache with 5ns access time and decrease of overall main memory miss rate to 0.5%, How many clock cycles miss penalty reduced?( Marks: 0.00 )
    1.   472.5
      Explanation:
      Given that CPI is 1.0
      5GHz machine, miss rate=2% 
      memory access time = 100 ns
      memory access time in cycles = 100ns/(1/5GHz)
            = 100ns/(0.2ns/clock cycle)
            = 500 clock cycles.
      Without L2 Cache:
      Miss penalty = 500 clock cycles.

      With L2 cache:
      L2 access time in clocks = 5ns/(1/5GHz) 
              = 25 clock cycles.
                          miss penalty =  25 + 0.5%(500)
               =  27.5 clock cycles.
      miss penalty reduced = 500 - 27.5 = 472.5 clock cycles.
    2.   473.5
