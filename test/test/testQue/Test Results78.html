  6. Which of the following IP addresses is matched by the address prefix 153.234.99.10/20?( Marks: 2.00 )
    1.   153.235.31.10
    2.   153.234.95.23
    3.   153.234.102.10
      Explanation:
      The last address is not even a valid IP address, so we can rule it out. (because 257 octet is not valid)
      The masks given as 20, so addresses must have the same first 20 bits for the prefix to be a match. i.e first 20 bits should be matched with the IP’s given in options.
      We can rule out 123.235.31.10 because of 2nd octet is differ.
      For the other two addresses, we need to look at how their 3rd octet can match with prefix(first 20 bits of given IP)
      The third byte of the prefix is of the form 0110 ****, i.e., only the first 4 bits matter given the mask size of 20 (8+8+4 = 20). We have: 95 = 0101 1111,
      102 = 0110 0110.
      Hence, the prefix is a match only for 123.234.102.10
    4.   153.257.140.10


  7. In an organisation, class B address is given by ISP. There are total 6 projects going in 6 different blocks and each project wants its own subnet to do experiments. The company has bought all the routers very long back so that routers are not upgraded with “ip subnet zero” command. If you are the network administrator, what is the best subnet mask to use in this scenario?
    Note: The routers configured with “ip subnet zero” are the only routers to support the usage of first(with all 0’s in subnet bits place) and last subnet(with all 1’s in subnet bits place)( Marks: 2.00 )

    2.   255.255.248.0
    3.   255.255.224.0
      Explanation:
      The default mask for a Class B is 255.255.0.0.
      Answer C extends that mask by three bits, creating 8 subnets (23 =8). Because of routers are not upgraded with “ip subnet zero” command the first subnet (all 0’s in subnet bits place) and last subnet(all 1’s in subnet bits place). So we are left with six subnets.
      Answer A is incorrect because it is the default mask for a Class B and not subnetted at all.
      Answer B and D are incorrect because although they create sufficient subnets, they do not maximize the number of hosts per subnet and so are not the best answer.

  10. Consider the following addresses as the first IP addresses of subnets in Oxford University. Among those starting IPs, 4 IP’s are given below and some of them are part of a block, whose size is 1024(i.e block have 1024 IP’s). Which of the following IPs is/are a part of the block of 1024 IPs?

  12. What could be the network mask if direct broadcast address of a network is 186.19.07.255?( Marks: -0.66 )
    1.   255.255.248.0
    2.   255.255.252.0
    3.   255.255.254.0
    4.   All the above
      Explanation:
      Without knowing how many subnets are there or how many bits are used for subnetting, we can not say the netmask perfectly.
      If we assume 3rd octet first 5 bits as subnet borrowed bits, mask will be 255.255.248.0. And network ID is 186.19.0.0
      If we assume 3rd octet first 6 bits as subnet borrowed bits, mask will be 255.255.252.0.
      And network ID is 186.19.4.0
      If we assume 3rd octet first 7 bits as subnet borrowed bits, mask will be 255.255.254.0.
      And network ID is 186.19.6.0
      All are possible, so All the above

      Underline bits are used to do subnetting.
      192.168.0.0/19 → 192.168.55.255/19 ⇒ PU(8192 possible host IP’s)
