    (Example: 011(110 = 6), 0, 0011 (1100 = 12), 1001 (1001 = 9), 01001(10010 = 18), 10101(10101 = 21), etc.)( Marks: 2.00 )
    1.   5
    2.   3
    3.   2
    4.   4
      Explanation:
      First, we find the DFA that accepts the set of all binary strings is divisible by 3.
      BN ≡ 1(mod 3)
      ∴There are 3 states and q0 is the final state.

      Now, for finding the reverse of a DFA, we must:
      1. Interchange the initial and final states.
      2. Reverse all the edges in the transition diagram.
      3. If more than one initial state occurs, then simplify initial state by adding ∊-transitions.
      4. This gives an NFA. Convert the NFA to DFA.

      Now, we need to handle the border cases. Given that ∊ does not corresponds to any number, so we need to reject it. So we add a new initial state that is not final, such that on reading any letter, it goes to the state to which q0 goes to on reading the same letter. i.e., on reading 0, q0 goes to q0. So on 0 qinit goes to q0. And q0 goes to q1 on reading 1, so qinit goes to q1 on reading 1.

      Note: Given the automaton constructed above (without handling ∊) is its own reversal in fact proves the following fact: any number written in binary is divisible by 3 if and only if the number corresponding to the reversal of the binary string is also divisible by 3.
