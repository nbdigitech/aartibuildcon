      CPIideal = (0.4*1)+(0.3*1.5)+(0.2*1.5)+(0.1*1)

      CPIactual = CPIideal + Memory Stalls per instruction
      The Memory stalls per instruction(Stalls/Inst) is given by

      CPIactual = 1.25 + 8
      = 9.25
