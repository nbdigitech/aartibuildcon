  7. Which of the following statements is/are False?

    II. The time complexity of Breadth First Search is always O(V+E) if you represent graph using either adjacency list or adjacency matrix.( Marks: -0.66 )
    1.   Only I
    2.   Only II
    3.   Both I and II
      Explanation:
      I. False: If the graph contains negative-weight cycle then no short path exists

      II. False: With an adjacency matrix representation, visiting each vertex takes O(v) time, as we must check all N possible outgoing edges in the adjacency matrix. Thus, BFS will take O(v2) time using adjacency matrix.

  11. Which of the following statements is/are False?

    I. If the DFS finishing time f[x] > f[y] for two vertices ‘x’ and ‘y’ in a directed graph G, and x and y are in the same DFS tree in the DFS forest, then ‘x’ is an ancestor of ‘y’ in the depth first tree.
    II. The topological sort of an arbitrary directed acyclic graph G=(V, E) can be computed in linear time.( Marks: -0.66 )
    1.   Only I
      Explanation:
      I. False: In a graph with 3 nodes s, x, and y, with edges (s, x) and (s, y) and s is a starting point for the DFS, x an y are siblings in the DFS Tree, neither as the ancestor of the other.

      II. True: A topological sort can be obtained by listing the nodes in the reverse order of the exit times produces by a DFS traversal of the graph. The DFS can also be used to detect if there is a cycle in the graph. The running time of DFS is O(V+E).

    3.   Both I and II
