  50. Consider a system with 2-levels of paging and a TLB with hit rate of 95% and TLB access time of 1ns. Find the effective memory access time if there’s a data cache whose hit rate is 85% and cache access time is 1ns, and main memory access time is 100ns.( Marks: 2.00 )

      pc= cache hit rate = 85% = 0.85
      Time to access the content from main memory = C + (1-pc) * (m) = 1+0.15*100 = 1 + 15 =16

      = 11 + 16 = 27ns
