  20. Consider a pipelined processor having 5 stages. The stage delays are 1, 1.4, 1.2, 3, 2ns and interstage buffer delays are 1ns. The 3rd stage is capable of deciding branch target address. Processor starts fetching new instruction when the conditional branch outcome is known. 25% of instructions are conditional branch, then calculate the execution time for 1200 instructions?( Marks: 0.00 )
    1.   6000ns
    2.   4800ns
    3.   7200ns
      Explanation:
      5 stage pipeline
      Delays are 1, 1.4, 1.2, 3, 2ns
      Inter stage buffer delays are 1 ns
      3rd stage is capable of knowing Branch Address
      25% instructions are branch instructions

      Number of clock cycles = 2
      Clock time= max stage delay + BUFF delay
      = 3 + 1 = 4ns
      Number of clock cycles = (1 + stall Freq * stall cycles) clocks
      = (1 + 0.25 * 2) clocks
      = 1.5 clocks
      Execution time = 1.5 * 4 ns / instruction
      = 6 ns / instruction
      Execution time for 1200 instructions = 6 ns / instruction * 1200 instruction
      = 7200 ns
    4.   5000ns

  22. Consider a system with main memory of size 64MB. If the no. of tag bits = 10, find the size of a 4-way set associative cache.( Marks: 0.00 )
    1.   256 KB
      Explanation:

      No of bits for set no. + no of bits for block offset = 16 bits

      Let no of bits for set no = x , no. of sets = 2x

      No of bits for block offset = 16 -x, cache block size = 2(16-x)Bytes

      In a 4 way cache no. of lines per cache = 4

      Cache size = No. of sets * No. of lines per set * size of cache block
      = (2x) * 4 * 2(16-x)
      = 4 * (216)
      = 218 = 256 KB
    2.   128KB
    3.   64KB
    4.   32KB
