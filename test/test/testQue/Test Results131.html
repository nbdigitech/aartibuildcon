    Gexp=({E,I},∑ ,P,E) where ∑= {a,b,0,1,(,),+,*} and P is

    The language h(L(Gexp))∩0*1* is( Marks: 0.00 )
    1.   Regular
    2.   Context free
    3.   Recursive but not context free
    4.   Deterministic context free language
      Explanation:
      We know that h(L(Gexp)) is context free and 0*1* is regular. From the closure properties, intersection of a context free language and a regular language is CFL.
      In this case, we need to check whether h(L(Gexp))∩0*1* is DCFL or not.
      h(L(Gexp))={ϵ,01,0011,0101,001011,…}
