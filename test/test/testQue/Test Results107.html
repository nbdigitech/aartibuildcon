  18. If a machine have a 48-bit address space with a page size of 16 KB and that page table entry takes 8 bytes. How many levels of page tables would be required to map the virtual address space if every page table is required to fit in a single page?( Marks: 0.00 )
    1.   2
    2.   4
      Explanation:
      There will be four levels of page tables.

      no. of entries in 1st level page table = (2^48)/2^14 = 2^34
      Size of 1st level page table = (2^34) * (2^3) = (2^37)

      Size of 1st level page table > size of page, (2^37 > 2^14). So we need next level of paging:
      no. of entries in 2nd level page table = (2^37)/2^14 = 2^23
      Size of 2nd level page table = (2^23) * (2^3) = (2^26)

      Size of 2nd level page table > size of page, (2^26 > 2^14). So we need next level of paging:
      no of entries in 3rd level page table = (2^26) / (2^14) = (2^12)
      Size of 3rd level page table = (2^12) * (2^3) = (2^15)

      Size of 3rd level page table > size of page, (2^15 > 2^14). So we need next level of paging:
      No of entries in 4th level page table = (2^15) / (2^14) = (2)
      Size of 4th level page table = (2) * (2^3) = (2^4)


      So the split will be [1, 11, 11, 11, 14]
    3.   3
    4.   1

  32. Linked Data Question 32 & 33:
    Consider a computer system has the segmented paging to implement virtual memory. The memory is byte addressable both virtual and physical address space have 2^16 bytes each. The virtual address space is divided into 8 equal segments. The memory management unit has a hardware segment tables. Each entry of which contains the physical address of the page table for segment. Page tables are stored in main memory and consist of 2 byte page table entries.

    What is the minimum page size in bytes so that the page table for a segment requires at most one page to store it? (Assume that the page size can only be a power of 2).( Marks: 0.00 )
    1.   128
      Explanation:
      Size of each segment = 2^16/8=2^13
      Let the size of page be 2^k bytes
      We need a page table entry for each page. For a segment of size 2^13, number of pages required will be
      2^13 − k and so we need 2^13 − k page table entries. Now, the size of these many entries must be less than or equal to the page size, for the page table of a segment to be requiring at most one page. So,
      (2^13−k)×2=2^k
      k=7
      So, page size =2^7=128 bytes
    2.   64
    3.   256
    4.   512
