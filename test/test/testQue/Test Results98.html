  5. If the expression ((A + B) * C + D * (E + F) * G) + H is evaluated, the value obtained is the same as the value of which of the following prefix expressions( Marks: 0.00 )
    1.   + * + + A B C * * D + E F G H
    2.   + + * + A B C * * D + E F G H
      Explanation:
      1. Given expression is ((A + B) * C + D * (E + F) * G) + H
      2. Evaluation (A+B) results (+ A B * C + D * (E + F) * G) + H
      3. Evaluation of (E+F) results (+ A B * C + D * + E F * G) + H
      4. Evaluation of D * + EF results in *D+EF so it becomes (+ A B * C + * D + E F * G) + H
      5. Evaluation of +AB * C results in *+AB C so it becomes (* + A B C + * D + E F * G) + H
      6. Evaluation of *D+EF * G results ** D+EFH so it becomes (* + A B C + * * D + E F G) + H
      7. Evaluation of *+ABC+ **D+EFG becomes + *+ABC**D+EFG + H
      8. Finally Evaluation of +*+ABC**D+EFG + H becomes + + * + A B C * * D + E F G H
    3.   + * + * A B C + + D * E F G H
    4.   * + + + A B C * * D + E F G H

  30. An array M[12][10] is stored in column major order, with each element occupies 4 bytes, and the address of M[7][8] is stored at the location 1020. Find out the address of M[10][10]. Assume array index start with ‘0’.( Marks: 0.00 )
    1.   1116
    2.   1128
      Explanation:
      As we know that if M = Total rows, N = Total columns
      Address A[p][q] = Base + W * [p + M * q],
      Array M[12][10], words per element = 4 bytes
      Given M[7][8] address = 1020
      ⇒ 1020 = Base + 4 * [(7 – 0) + 12(8 – 0)]
      1020 = Base + 4 * [7 + 96]
      1020 = Base + 4 * 103
      Base = 1020 – 412 = 608
      Now, to find M[10][10]
      ⇒ 608 + 4 * [(10 – 0) + 12(10 – 0)]
      608 + 4 * [10 + 120]
      608 + 4 * 130
      608 + 520
      M[10][10] = 1128
    3.   1126
    4.   1118
