  23. A binary search tree contains the numbers 1, 2, 3, 4, 5, 6, 7, 8. When the tree is traversed in pre-order and the values in each node printed out, the sequence of values obtained is 5, 3, 1, 2, 4, 6, 8, 7. If the tree is traversed in post-order, the sequence obtained would be( Marks: 0.00 )
    1.   8, 7, 6, 5, 4, 3, 2, 1
    2.   1, 2, 3, 4, 8, 7, 6, 5
    3.   2, 1, 4, 3, 6, 7, 8, 5
    4.   2, 1, 4, 3, 7, 8, 6, 5
      Explanation:
      In post order traversal we have to traverse in Left Right Root recursively,

      Method 1:
      Step 1 : The first element in the pre-order traversal is the root of the tree. Elements less than the root form the left sub-tree. Elements greater than the root form the right sub-tree.

      Step 2: Find the structure of the left and right sub-trees using step 1 with a pre-order traversal that consists of the elements we worked out to be in that sub-tree placed in the order they appear in the original pre-order traversal.

      Step 3: Traverse the resulting tree in post-order to get the post-order traversal associated with the given pre-order traversal.

      Method 2:
      As it is a binary search tree, we can easily construct tree using preorder I.e., Root Left Right using 5, 3, 1, 2, 4, 6, 8, 7

      1. 5 is the root element, 3 (<5) so it will be left child
      2. Next is 1(<3), so it will be left child to 3
      3. Next is 2(>1), so it will be right child to 1
      4. Next is 4(>3), so it will be right child to 3
      5. Next is 6(>5), so it will be right child to 5
      6. Next is 8(>6), so it will be right child to 6
      7. Next is 7(<8), so it will be left child to 8

      In post order traversal method, first we traverse left subtree, then right subtree and finally root
      so 2 1 4 3 7 8 6 5 will be printed.

  25. The height of a tree is the length of the longest root-to-leaf path in it, root is at height 0. The sum of the maximum and minimum number of nodes in a binary tree of height 9 is ________.( Marks: 0.00 )
    1.   520
    2.   1032
    3.   521
    4.   1033
      Explanation:
      The maximum number of nodes in the binary tree of height h = 2^(h+1) – 1
      So, the maximum number of nodes in the given binary tree of height 9 = 2^10 – 1 = 1023
      and for minimum number of nodes, there can be only one node at each level. So, the minimum number of nodes is 10
      The sum of the maximum and minimum number of nodes in a binary tree of height 9 is 1023+10 = 1033.
