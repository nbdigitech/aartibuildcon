  4. Which of the following statements is true?( Marks: 1.00 )
    1.   “Does a DFA recognize the empty language?” is undecidable
    2.   “Does a TM M accepts a string w?” is decidable
    3.   “Does a CFG G generate a string w?” is undecidable
    4.   “Does a TM M recognize the empty language?” is undecidable
      Explanation:
      Almost all properties of regular languages are decidable: emptiness, finiteness, membership, equalness all are decidable. So, the question “Does a DFA recognize the empty language?” is decidable.

      The question “Does a CFG G generate a string w?” is decidable. This is membership property which is decidable for context-free languages.

      Almost all properties of TM are undecidable. Acceptance property of TM is undecidable.

      “Does a TM M recognize the empty language?” is undecidable. This is Blank Tape Acceptance Problem (BTAP), which is undecidable. We can prove the undecidability of BTAP by reducing the Halting problem, which is undecidable, to BTAP, and prove by contradiction.

  7. Which of the following statements is/are CORRECT?
    (i) Every Turing-decidable language is also context-free
    (ii) There exists a language that is recognized by the nondeterministic Turing machine that is not recognized by the deterministic Turing machine
    (iii) If a language L is Turing decidable then it is also Turing recognizable.( Marks: 2.00 )
    1.   (i) – False, (ii) – True, (iii) – False
    2.   (i) – True, (ii) – False, (iii) – False
    3.   (i) – False, (ii) – False, (iii) – True
      Explanation:
      (i) Consider the language {an bn cn |n ≥ 0}. This is decidable but not context-free. So, (i) is false.
      (ii) Every nondeterministic Turing machine has an equivalent deterministic Turing machine. They have and equal acceptance power. So, there does not exists a language that is recognized by the nondeterministic TM but not by its deterministic version. So, this is false.
      (iii) A language is decidable iff there is a Turing Machine which will accept strings in the language and reject strings not in the language.
      A language is recognizable iff there is a TM which will halt and accept only the strings in that language and for strings not in the language, the TM either rejects, or does not halt at all.
      If a language L is Turing decidable then it is Turing recognizable, but the converse is not true.

  11. If L and L'are decidable, then which of the following is/are true?
    L ∪ L' is decidable
    L ∩ L' is decidable
    L - L' is decidable
    L' - L is decidable( Marks: 2.00 )
    1.   Only (i) and (ii) are true
    2.   All are true
      Explanation:
      L ∪ L' is decidable
      :- Let M and M' be Turing machines that decide L and L', respectively. We can build a Turing machine M∪ that decides L∪L'as follows. First, M∪ copies its string w onto another tape. Then M∪ runs M on the input string w (on the first tape), and then runs M'on input w on the second tape. If either M or M'accepts, then M∪ accepts; if both M and M' reject, then M∪ rejects. So, L∪L' is decidable.

      L ∩ L' is decidable
      :- Let M and M' be Turing machines that decide L and L', respectively. We can build a Turing machine M∩ that decides L ∩ L'as follows. First, M∩ copies its string w onto another tape (a second tape). Then M∩ runs M on the input string w on the first tape, and then runs M'on input w on the second tape. If both M and M'accepts, then M∩ accepts; if either M or M' reject, then M∩ rejects. So, L ∩ L' is decidable.

      L - L' is decidable
      :- L - L' can be written as L ∩ (L' )'=L ∩ L = L, which is decidable. So, L - L' is decidable.

      L' - L is decidable
      :- L' - L can be written as L' ∩ L' = L', which is decidable. So, L' - L is decidable.
    3.   Only (iii) and (iv) are true
    4.   Only (i), (ii) and (iii) are true


  12. Consider the languages
    L1 = {babaabaaab…ba(n-1)banb | n ≥ 1}
    L2=Complement(L1)
    Which of the following is TRUE?( Marks: -0.66 )
    1.   Both L1 and L2 are CFLs
    2.   L1 is CFL but L2 is not CFL
    3.   Neither L1 nor L2 is CFL
    4.   L1 is not CFL but L2 is CFL
      Explanation:
      The language L1 is not CFL. We can’t build a PDA for L1: if we count the first group of a’s then we’ll need to pop them to match against the second group. But, then what do we do for the third group and the rest of the groups? So, it is not CFL.

      L2 is CFL. Since L2 is complement of L1, we do not need to check all the groups of a follow the right pattern. Instead, we simply have to find a group that does not follow the pattern. We can do that with a nondeterministic PDA P. P goes through the string making sure that the basic b(a+ b)+ structure is followed. Also, it non-deterministically chooses one group of a’s to count. It then checks that the following group does not contain one more ‘a’ than the current group. If any branch succeeds (i.e., it finds a mismatched pair of adjacent a groups) then P accepts.

      We can also show that L2 is CFL by decomposing the language into union of two simpler languages.
      i.e., the complement of L1 contains the strings of the form L11 ∪ L12, where
      L11 = {(a ∪ b)* - b(a+ b)+ }
      L12 = {xbam ban by | x,y ∈ {a,b}* and m + 1 ≠ n}

      The language L11 includes all the strings which do not follow the pattern b(a+ b)+. It is easy to see that L11 is regular.
      The language L12 includes all the strings which follows the pattern but do not follow the rule that each group of a’s must contain one more ‘a’ than its predecessor. We can construct a PDA for L12 which is much similar to the PDA of {an bm | n ≠ m}. So L12 is CFL.
      The union of regular and CFL is CFL. So L2 is CFL.

  15. Which of the following statements is/are true?
    (i) Regular languages are closed under inverse homomorphism
    (ii) Context free languages are closed under complementation but deterministic context free languages are not.
    (iii) Halting problem of recursive languages is undecidable.
    (iv) Membership property of TM is undecidable.( Marks: 2.00 )
    1.   (i) – T, (ii) – F, (iii) – (F), (iv) – T
      Explanation:
      (i) Regular languages are closed under inverse homomorphism
      :- Regular languages are closed under homomorphism and inverse homomorphism. So, this is TRUE

      (ii) Context free languages are closed under complementation but deterministic context free languages are not.
      :- CFLs are not closed under complementation but DCFLs are. So, this is FALSE.

      (iii) Halting problem of recursive languages is undecidable.
      :- The Halting Problem can be defined as: given a Turing machine and an input to the Turing machine, does the Turing machine finish computing in a finite number of steps (a finite amount of time)?
      Remember, recursive languages are decidable. Hence, the Halting Problem is decidable for recursive languages. So, this is FALSE.

      (iv) Membership property of TM is undecidable.
      :- Membership property of TM is undecidable. In fact, most of the properties of TM are undecidable. So, this is TRUE.
    2.   (i) – F, (ii) – F, (iii) – (F), (iv) – F
    3.   (i) – F, (ii) – T, (iii) – (F), (iv) – F
    4.   (i) – T, (ii) – T, (iii) – (T), (iv) – T
