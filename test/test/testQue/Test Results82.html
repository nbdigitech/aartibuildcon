  12. If ExOR operation is performed among all bits of signed number, which is stored in a 4-bit register results as ‘1’ what is value results when all the bits of the same number undergo ExOR operation when stored in 8 bit register?( Marks: 0.00 )
    1.   0
    2.   1
      Explanation:
      Given ExOR of bits of 4 bit signed number is 1
      Eg:- (-3) = 1101 ⇒ 1 ⊕ 1 ⊕ 0 ⊕ 1 = 1
      (-2) = 0010 ⇒ 0 ⊕ 0 ⊕ 1 ⊕ 0 = 1
      If this number is stored in 8 bit register, then sign extension happens.
      This ends up in 4 more 1’s or 4 more 0’s in the number.
      Eg: (-3) = 1111 1011
      2 = 0000 0010
      This can’t change the value of ExOR among bits
      (-3) = 1111 1101 ⇒ 1 ⊕ 1 ⊕ 1 ⊕ 1 ⊕ 1 ⊕ 1 ⊕ 0 ⊕ 1
      2 = 0000 0010 ⇒ 0 ⊕ 0 ⊕ 0 ⊕ 0 ⊕ 0 ⊕ 0 ⊕ 1 ⊕ 0
      So, the answer is 1.
    3.   10
    4.   11
