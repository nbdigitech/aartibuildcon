  10. How many 3 states DFA’s with designated initial state can be constructed over Σ = {a, b}?( Marks: 0.00 )
    1.   5832
      Explanation:
      Σ={a,b}
      Q={q0,q1,q2 }
      Let q0 be the initial state.

      ∴8 types of DFA’s possible for q0 as initial state.
      In each FA, there can be 3 possible transitions from each state over each symbol.

      ->This gives 36 combinations.
      ∴Total no. of DFA’s = No. of types × No. of DFA’s in each type = 8 × 36 = 5832.

      Note:
      If |Σ| = m, |Q| = n, then
      1. Number of types of DFA's is 2n
      2. Number of DFA's of each type = nmn
      3. Total number of DFA's = 2n.nmn
