  7. Consider the function f(a,b,c)=ab'+c'. Which of the following statements is true?( Marks: 2.00 )
    1.   f is not functionally complete
    2.   f is partially functionally complete by using ‘1’
    3.   f is partially functionally complete by using ‘0’
    4.   f is functionally complete
      Explanation:
      f(a,b,c)=ab'+c'
      f(a,a,a)=aa'+a'=a'⟹NOT

      =aa+c=a+c⟹OR
      ={OR,NOT}
      Hence, f is functionally complete.


  8. Given x,y,z are Boolean variables and f(x,y,z)=f(y',x',z). How many such functions are possible with x, y, z?( Marks: 2.00 )

      =y' x' z+xyz
      =f(x,y,z)
      We can call (xyz),(y' x' z) are terms on whose presence the function hold the property.
      Let f(x,y,z)=xy'z'
      f(y',x',z)=y' (x' )' z'
      =y' xz'
      =xy' z'
      =f(x,y,z)
      ⟶ The term xy'z' holds the property.
      ⨳ So, the function should be formed with the above kind of couple terms and single terms.
      ⨳ So, we need to find all such terms.


      (0, 0, 0), (1, 1, 0) ⇒ (0, 6)
      (0, 0, 1), (1, 1, 1) ⇒ (1, 7)

      The couple terms (0, 6) collectively hold the property
      The couple terms (1, 7) collectively hold the property
      The term 2 alone holds the property
      The term 3 alone holds the property
      The term 4 alone holds the property
      The term 5 alone holds the property
      For each of the above we have two choices, either to add the term into function or not.
      (0, 6) (1, 7) 2 3 4 5
      2 x 2 x 2 x 2 x 2 x 2

      ⇒ 26
