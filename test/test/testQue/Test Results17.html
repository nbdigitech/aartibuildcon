  23. How many four digit numbers can be formed using the digits 1, 2, 4, 6, 7 and 9 such that each number is divisible by 3 but not by 9? (Repetition of digits is not allowed)( Marks: 0.00 )
    1.   84
    2.   48
    3.   72
      Explanation:
      For a number to be divisible by 3 but not by 9, the sum of the digits of number should be 6 or 12 or 15 or 21 or 24(in simple words the sum of digits should not be a multiple of 9).
      We exclude the cases – where sum of the digits will be 9 and 18.
      Maximum possible sum of the given digits is
      9 + 7 + 6 + 4 = 26
      As repetition is not allowed, we must be one digit only once.
      The favorable cases are
      9, 7, 4, 1 – 4!
      9, 6, 4, 2 – 4!
      9, 7, 6, 2 – 4!
      ∴ The number of such 4-digit numbers = 4! × 3 = 72
      (Note: No such number can be formed where the sum of the digits is 6, 12 or 15)

  31. Which of the following is/are valid arguments?
    A1: [p→r,r→s,t∨~s,~t∨u,~u]⇒~p
    A2: [p→q,q→r,s→~r,p]⇒~s( Marks: 0.00 )
    1.   Only A1
    2.   Only A2
    3.   Both A1 and A2
      Explanation:
      The given premises are
      (1) p→r
      (2) r→s
      (3) t∨~s
      (4) ~t∨u
      (5) ~u
      (6) s→t (from ③)
      (7) t→u (from ④)
      (8) p→s (from ① and ② using transitive)
      (9) s→u (from ⑥,⑦ using transitive)
      (10) p→u (from ⑧ and ⑨ using transitive)
      (11) ~p (from ⑤ and ⑩ using Modus tollens)
      The conclusion ~p is derived from the given premises. So, A1 is valid.
      A2: The given premises are
      (1) p→q
      (2) q→r
      (3) s→~r
      (4) p
      (5) p→r(from ①,② using transitive)
      (6) r (from ④,⑤ using Modus Ponens)
      (7) ~s (from ③,④ using Modus tollens)
      The conclusion ~s is derived from the given premises. So, A2 is valid.
    4.   Neither A1 nor A2
