  30. Common Data Questions Q30 & Q31:

    Consider the system with the 2-level paging with TLB support. The page table is divided into 2K pages each of size 4K words. If the physical address space has 64M words. TLB access time is 25ns and main memory access time is 100ns. The CPU finds 145 page references in the TLB out of total of 180 references.
    Find the effective memory access time if there's a page fault for every 1 million memory accesses and page fault service time is 10ms?( Marks: 0.00 )
    1.   164ns
    2.   174ns
      Explanation:
      Page fault rate, PFR = 1 / (10^6)
      Page fault service time, PST = 10ms = 10 * (10^6) ns

      The CPU finds 145 page references in the TLB out of total 180 page references, so the hit rate, p = 145 / 180 =0.805

      TLB access time, t = 25ns,
      Main memory access time , m = 100ns
      k = no. of levels of paging = 2

      The effective memory access time includes the time for Virtual Address to Physical Address conversion and once the physical address is obtained the time to access the content.

      VA -> PA conversion time = t + (1-p)(km) = 25 + (0.195) * 2*100 = 25 + 39 = 64ns

      Time to access the content: The content will be searched first in the main memory and if there's a page fault then the page fault service time to be included.
      Content accessing time = PFR (PST + m) + (1-PFR)m = PFR * PST + PFR * m + m – PFR *m = PFR * PST + m
      =(1 / (10^6) )* (10 * 10^6) + 100
      =10 + 100 = 110

      Effective Memory Access Time = 64 + 110 = 174ns
    3.   125ns
    4.   150 ns
