  3. many reflexive and anti-symmetric relations are there on a set with 4 elements?( Marks: 0.00 )
    1.   243
    2.   282
    3.   627
    4.   729
      Explanation:
      The reflexive relation must contain all the diagonal elements.
      {(1,1),(2,2),(3,3)…(n,n) }
      Every superset of a reflexive relation is also a reflexive relation.
      In the anti-symmetric relation, (x, y) is present in the relation and (y, x) is also present in the relation iff and only if x = y.
      ∴The smallest relation which is both reflexive and anti-symmetric is all the diagonal elements of a set.
      ∴ The diagonal elements are compulsory in a relation which is both reflexive and anti-symmetric relation.
      The remaining non-diagonal elements are n^2-n.
      In the non-diagonal element if we have (x, y) in the relation then we don’t have (y, x) in the relation. From the pair (x, y)and (y, x) will have either one (or) none in the relation.
      Total such pairs are (n2-n)/2.
      Each pair is having 3 possibilities (x, y) or (y, x) or none.
      So, total number of relations which are both reflexive and anti-symmetric on a set with ‘n’ elements =3((n[sup]2-n)[/sup]/2)
      ∴The total number of reflexive and anti-symmetric relations on a set with 4 elements =3(4(4-1)/2)
      =36
      =729

  23. A relation over the set S={a,b,c} is defined by: {(a,a),(a,b),(b,a),(a,c),(b,c),(b,b),(c,c) } what properties hold for this relation?
    I. Symmetric
    II. Reflexive
    III. Anti-symmetric
    IV. Irreflexive( Marks: 0.00 )

    2.   II only
      Explanation:
      The given set S={a,b,c}
      The given relation R on set S={(a,a),(a,b),(b,a),(a,c),(b,c),(b,b),(c,c) }
      The elements: (a,a),(b,b),(c,c) is present in the relation R. So, R is reflexive. If R is reflexive then R is not irreflexive.

      Both the elements (a, b) and (b, a) are present in the relation. So, the R is not anti-symmetric.

    4.   III and IV only
