    2.   5/18
      Explanation:

    2.   O(n2)
    3.   O(m*n)
      Explanation:
      Algorithm:
      (i) Scan the input sets one by one
      (ii) For each element keep track of the counter. Counter indicates the frequency of the occurrences in all the sets
