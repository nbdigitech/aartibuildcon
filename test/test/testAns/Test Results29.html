  60. Common Data Question
    Consider a TCP session from a client A to a server B. Suppose that the first packet from A, following the “SYN packet” has a sequence number of 200.     1.   1179 bytes
    2.   980 bytes

  61. 980 bytes. The first data packet has a sequence number of 200. This would be the ‘random’ sequence number chosen. The later packet with sequence number 700 and IP payload length 500 would include the TCP header in the 500 bytes, so there are 480 bytes of TCP data including bytes numbered 700 through 1179. So, including this later packet, A would end up sending bytes numbered 200 through 1179 for a total of 980 bytes
    What sequence number would you expect in the next packet from A?( Marks: 2.00 )
    1.   1179
    2.   981
